# Why?
The goal of this library is to take away the pain of manually handling fetching states (idle, loading, error)
and their visual feedback. Just hand over an HTTP request to the various hooks this library offers, and let it handle the
visual feedbacks and the what-ifs.  
It reduces boilerplate but also give you a wide range of customization tools for general usage.

#### 2. Wrap your content around the `Fetcher` component
```tsx
<div>
  <Fetcher refs={ref}>
    My content
  </Fetcher>
</div>
```

#### 3. Use the fetcher reference to perform HTTP or custom requests
```ts
ref.fetch(...);
// or
getItem(1);
```

# Installation

```bash
yarn add react-fetcher-hooks
npm install react-fetcher-hooks
```

Make sure to add the library CSS file to your build:
```ts
import 'react-fetcher-hooks/dist/styles.css';
```
